# 스테이지 1 : 관 돌 부수기

> **기능 기획서** · 이집트 투탕카멘 유물 도굴 게임

제한 시간 안에 관(棺)을 덮은 돌을 모두 부수되, 그 아래 드러나는 유물(관)은 손상시키지 않아야 하는 스테이지. 빠르지만 위험한 **드릴**과 느리지만 안전한 **끌** 사이의 긴장감이 핵심이다.

## 1. 도구 2개 확정

빠르게 깨는 도구와 섬세하게 마무리하는 도구, 두 가지를 제공한다. 플레이어는 상황에 따라 두 도구를 전환하며 사용한다.

| 구분 | 드릴 | 끌 |
| --- | --- | --- |
| **용도** | 큰 돌 덩어리 빠르게 제거 | 관 주변 잔여 돌 섬세하게 제거 |
| **인터랙션** | Pressure 꾹 누르는 동안 연속 작동 | Pressure 누를 때마다 1회 타격 |
| **햅틱** | 드르르르륵 (연속 진동, 긴/짧은 주기) | 툭 ··· 툭 ··· (타격마다 단발) |
| **위험성** | 관 영역 진입 시 즉시 게임오버 | 관 영역 진입 시 즉시 게임오버 |
| **Pressure 반응** | 세게 누를수록 HP 감소 속도 빠름 | 타격마다 고정 HP 감소 |

**드릴 햅틱 상세**

- 살살 누름 → 긴 주기 진동 (드르··르··르) → 천천히 깨짐
- 세게 누름 → 짧은 주기 진동 (드르르르르르) → 빠르게 깨짐
- Pressure 값을 햅틱 주기에 연속적으로 매핑

## 2. 조각 시스템

바위를 미리 N개 조각으로 분할해 에셋으로 준비한다. 각 조각은 독립적인 HP를 가진다.

| HP 단계 | 상태 |
| --- | --- |
| **HP 100%** | 멀쩡한 돌 이미지 |
| **HP 66%** | 금이 간 이미지 (1단계) |
| **HP 33%** | 심하게 금이 간 이미지 (2단계) |
| **HP 0%** | 조각 소멸 → 아래 관 이미지 노출 |

- HP 단계마다 균열 이미지 교체로 시각 피드백 처리
- 조각이 소멸한 자리 = 관 노출 영역으로 즉시 등록
- 관 노출 영역에 도구 접촉 시 → 즉시 게임오버
- 구현 단순: 조각별 독립 관리, 파티클 없어도 균열 이미지만으로 충분

## 3. 플레이 흐름

1. **Hover로 드릴 조준**
   - 펜슬 가까이 대면 드릴 UI가 펜슬 따라 등장
2. **드릴로 큰 돌 덩어리 빠르게 제거**
   - Pressure 누르는 동안 드르르르륵, 손 떼면 멈춤
   - 세게 누를수록 HP 빠르게 감소 / 관 윤곽이 서서히 드러남
3. **관 영역이 드러나기 시작하면 끌로 전환 (UI 버튼 탭)**
   - 드릴을 드러난 관 위에 사용하면 즉시 게임오버
4. **끌로 남은 잔여 조각 섬세하게 제거**
   - 누를 때마다 툭 ··· 툭 ··· / 천천히 깨짐
   - 관 영역만 피하면 실패 없음
5. **제한 시간 안에 돌 전부 제거 → 클리어**
   - 관 뚜껑 완전히 드러남 → 다음 스테이지(자물쇠 따기)로 전환

### 플로우 차트

## 4. 클리어 / 실패 조건

### ✅ 클리어

- 제한 시간 안에 모든 돌 조각 HP를 0으로 만들기

### ❌ 실패 ① — 관 손상

- 드러난 관 영역 위에서 드릴 또는 끌 사용 시 즉시 게임오버
- "유물 파손" 연출 후 스테이지 처음부터 재시작

### ❌ 실패 ② — 시간 초과

- 제한 시간 안에 돌을 전부 제거하지 못하면 스테이지 처음부터 재시작

> 💡 **핵심 딜레마** : 드릴을 써야 빠른데, 관이 드러날수록 드릴 쓰기 위험해진다. 끌은 안전하지만 시간이 부족하다. 이 긴장감 하나로 게임성 완성.

## 5. 시각적 피드백

| 요소 | 내용 |
| --- | --- |
| **돌 균열** | HP 단계마다 이미지 교체 (멀쩡 → 금간 → 심하게 금간 → 소멸) |
| **관 드러남** | 조각 소멸 시 아래 관 이미지 노출 (SKSpriteNode 레이어 구조) |
| **위험 경고** | 드릴이 관 영역 근처 접근 시 Hover로 경고 표시 |
| **도구 전환 버튼** | 화면 한켠 고정 / 탭으로 즉시 전환 |
| **게임오버 연출** | 관 파손 시 화면 붉게 번쩍임 + 강한 햅틱 |

## 6. 사용 기술 정리

### PencilKit 기술

| 기술 | 사용 위치 |
| --- | --- |
| **Hover** | 드릴 UI가 펜슬 따라 이동 (조준 단계) |
| **Pressure** | 드릴 연속 진동 세기 / 끌 타격 감지 |
| **Haptic (연속)** | 드릴 사용 중 드르르르륵 진동 |
| **Haptic (단발)** | 끌 타격마다 툭 단발 진동 |

### SpriteKit 기술

| 기술 | 사용 위치 |
| --- | --- |
| **SKSpriteNode 레이어** | 돌 조각 / 관 이미지 레이어 분리 |
| **조각별 HP 관리** | 각 조각 노드에 HP 값 부여, 단계별 이미지 교체 |
| **충돌 판정** | 관 노출 영역 위 도구 접촉 감지 |
